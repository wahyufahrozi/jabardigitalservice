## Build Setup

1. Clone repository`$ git clone https://github.com/wahyufahrozi/jabardigitalservice`

2. Install depedencies

```bash
# with npm
$ npm install

# or with yarn
$ yarn install
```

3. Start Web server

```bash
$ npm start
```
